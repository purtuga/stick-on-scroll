Example Demo: Use of Headers with margin set

Demostration of stick on scroll of elements that have a margin set. In those cases, the offset from the top of the page will use that margin value and thus the element will be positioned using the edge of it and not the content's edge.

In addition, although these examples have setParentOnStick set to True, that input paramter is forced to False when the parent element is the <body> tag.
