jQuery.stickOnScroll.js Test

This page tests making element sticky when their imediate offset parent is NOT the viewport. In these cases, the position of the element can not be used because reports the TOP as relative to the first offset parent.

Example 1 | Example 2 | Example 3 | Example 4 |

Page Content

Use the container below to test the elements inside of it and ensure they are "sticky" correct. the element that are center aligned are inside a relative container and thus their offset is calcualted differently.

There are also elements that can be hidden/shown to see how the plugin handles changes to element's position on the page.

This area demonstrates the section getting "sticky" even if its first positioned parent (one that has a css position set) is not the viewport. See issue #3 on github.
